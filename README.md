# bookkeeping
简易的记账系统
一、项目概述
 
本项目是一款轻量级本地记账管理系统，基于 Python 语言开发，采用 JSON 文件实现数据持久化存储，可直接在本地运行，支持多用户权限分离的记账、统计和预算管理等核心需求。
 
二、开发语言及核心语法、特性
 
1. 开发语言：Python

2. 核心语法与特性

- 数据结构：使用字典（dict）存储用户信息、单条记账记录的属性（金额、类型、日期等），使用列表（list）存储多条记账记录、用户列表和消费类型列表，通过键值对和索引快速访问、修改数据。

- 文件操作：利用 Python 内置的  json  模块，实现  users.json  和  types.json  文件的读写操作（ json.dump()  写入数据、 json.load()  读取数据），完成数据持久化，保证程序重启后数据不丢失。

- 条件判断与循环：通过  if-elif-else  实现用户权限校验、功能分支选择，使用  while  循环构建程序的交互式菜单，支持用户多次操作。

- 函数封装：将注册、登录、记账、统计等功能封装为独立函数，实现代码模块化，提高可读性和复用性。

 
三、核心功能
 
1. 用户管理：支持普通用户注册、登录，区分管理员与普通用户权限；管理员可修改用户密码、维护消费类型，普通用户数据相互隔离。

2. 记账功能：实现收支记录的增、删、改、查，每条记录包含金额、类型、日期、备注等关键信息。

3. 统计功能：自动计算用户总余额，支持按消费类型统计收支金额，统计各类消费的记录次数。

4. 预算管理：用户可设置消费预算，并实时查看剩余预算金额，辅助控制消费。

5. 数据持久化：所有用户数据、记账记录、消费类型均存储在本地 JSON 文件中，无需额外配置，即开即用。
